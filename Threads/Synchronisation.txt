synchronized is a keyword in Java used to control access to a shared resource in multithreading.
It ensures that only one thread at a time can execute a block of code or method, preventing data inconsistency.

âš¡Why use synchronized?
To make code thread-safe
To prevent race conditions (multiple threads modifying the same data at the same time)
To ensure correct results when multiple threads work on shared resources
