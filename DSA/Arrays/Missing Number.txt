Example 1:
Input: nums = [3,0,1]
Output: 2
Explanation:
n = 3 since there are 3 numbers, so all numbers are in the range [0,3]. 2 is the missing number in the range since it does not appear in nums.

class Solution{
public int missingnumber(int[] nums){
int n = nums.length;
int total = n*(n+1)/2;
int actualsum = 0;
for(int num : nums){
actualsum+= num;
}
return total-actualsum;
}
Output:
2
